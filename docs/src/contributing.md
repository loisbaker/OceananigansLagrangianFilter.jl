# Contributing to OceananigansLagrangianFilter

We welcome contributions! These include issue reports, bug fixes, and feature implementations.
Contributions are reviewed on Github via pull request; to get started, fork the repository, make changes, and issue a pull request.
You can also contribute by submitting an issue.

## Issue reports 

If you find a bug or unexpected behaviour, please create an issue on [github](https://github.com/loisbaker/OceananigansLagrangianFilter.jl).
Please provide as much detail as possible, including the version of OceananigansLagrangianFilter and Oceananigans, your platform and architecture (CPU/GPU), and a minimal working example that demonstrates your problem as simply as possible. 

## Proposing features

You can propose new features by creating an issue on [github](https://github.com/loisbaker/OceananigansLagrangianFilter.jl).

## Contributing code

Code contributions, from fixing typos to implementing new features, are very welcome! If you'd like to contribute, just fork the repository, make changes, and issue a [pull request](https://github.com/loisbaker/OceananigansLagrangianFilter.jl/pulls). If you'd like to discuss a potential modification or use-case first, feel free to get in touch at lois.baker [at] ed.ac.uk.
